Базовые концепции Scala

I. Выражения (expressions)

Выражение - это вычислимое утверждение: 
                        1 + 1
Результат вычислений можно вывести при помощи println: 
                        println(1) // 1
                        println(1 + 1) // 2
                        println("Hello!") // Hello!
                        println("Hello," + " world!") // Hello, world!

I. Выражения (expressions)

Объявление значений (values): 
                        val x = 1 + 1
                        println(x) // 2

                        x = 3 // не компилируется
                        val y: Int = 41 - 1
Объявление переменных (variables): 
                        var x = 1 + 1
                        x = 3 // компилируется

II. Функции (functions)

Объявление функции: 
                        val add = (x: Int, y: Int) => x + y
                        println(add(1, 2)) // 3

                        val getTheAnswer = () => 42
                        println(getTheAnswer()) // 42

III. Методы (methods)

Объявление метода: 
                        def add(x: Int, y: Int): Int = x + y
                        println(add(1, 2)) // 3
Метод может принимать множественные списки параметров (каррирование): 
                        def addThenMultiply(x: Int, y: Int)(multiplier: Int): Int = {
                            val sum = x + y
                            sum * multiplier
                        }
                        println(addThenMultiply(1, 2)(3)) // 9

IV. Условное выражение (if expression)

Синтакисис аналогичен Java/C#/C++: 
                        if (a == b) doSomething()

                        if (trueOrFalse) {
                            doSomething()
                        } else if (a != b) {
                            doSomethingElse()
                        } else {
                            doAnotherThing()
                        }
Результат можно сохранить: 
                        val minValue = if (a < b) a else b

V. Циклы (for loops)

Перебор элементов в коллекции: 
                        val strings = Seq("A", "B")
                        val numbers = Seq(1, 2, 3)

                        for (str <- strings) print(s"$str; ") // out: A; B;

                        for (num <- numbers) print(s"$num; ") // out: 1; 2; 3;

                        for { str <- strings
                              num <- numbers } print(s"$str$num; ") // out: A1; A2; A3; B1; B2; B3;
Для классов коллекций также доступен foreach: 
                        val cars = Seq("Mercedes", "BMW", "Cadillac")
                        cars.foreach(println) // out: Mercedes BMW Cadillac

                        car => println(car)

V. Циклы (for loops)

Использование генераторов в циклах for: 
                        for (i <- 1 to 5) print(s"$i; ") // out: 1; 2; 3; 4; 5;

                        for (i <- 1 until 5) print(s"$i; ") // out: 1; 2; 3; 4;
Использование for для Map: 
                        val ingredients = Map(
                            "sugar"  -> 30,
                            "salt"   -> 5,
                            "pepper" -> 10
                        )

                        for ((title, amount) <- ingredients)
                            println(s"ing $title: $amount g") // out: ing sugar: 30 g ing salt: 5 g ing pepper: 10 g

V. Циклы (for loops)

Использование "гардов" (guards) в for-выражениях (for comprehensions): 
                        for { i <- 1 to 5
                              if (i % 2 == 0)
                        } print(s"i; ") // out: 2; 4;

                        for { i <- 1 to 5
                              if (i % 2 == 0)
                              if (i > 2)
                        } print(s"$i; ") // out: 4;
Конструкция for (enumerators) yield e в for-выражениях (for comprehensions): 
                        val strings = List("A", "B")
                        val res =
                            for { i <- 1 to 5 if i % 2 == 0
                                  str <- strings
                            } yield s"$str$i; "
                        res.foreach(print) // out: A1; B1; A2; B2;

V. Циклы (while/do-while)

Синтаксис цикла while: 
                        var i = 0
                        while (i < 5) {
                            print(s"$i; ")
                            i += 1
                        } // out: 0; 1; 2; 3; 4;
Синтаксис цикла do-while: 
                        var j = 2
                        do {
                            print(s"$j; ")
                            j -= 1
                        } while (j > 0) // out: 2; 1;
Возвращает тип Unit: 
                        var i = 0
                        val res = while (i < 5) {
                            someFunction(i)
                            i += 1
                        }
                        print(res) // out: ()
